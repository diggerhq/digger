# Atlas Migrations

This directory contains database migration files for all supported database types.

## Structure

```
migrations/
├── postgres/   # PostgreSQL migrations
├── mysql/      # MySQL migrations
├── sqlite/     # SQLite migrations
└── sqlserver/  # SQL Server migrations
```

## Generating Migrations

From the `taco/` directory:

```bash
# Generate migrations for all databases
make atlas-diff-all name=my_change_description

# Or generate for a specific database
make atlas-diff-postgres name=my_change
```

## Migration Files

Migration files are automatically generated by Atlas based on GORM model changes in `internal/query/types/models.go`.

Each migration file follows the naming pattern: `YYYYMMDDHHMMSS_description.sql`
